Introduction

Python is a high-level, interpreted programming language known for its simplicity and readability. It supports multiple programming paradigms, including procedural, object-oriented, and functional programming.

Getting Started with Python

Installation

To install Python, visit the official Python website and download the latest version. Python comes with a built-in package manager called pip, which makes installing additional libraries easy.

Steps to install:

  1. Download the Python installer.
  2. Run the installer and check the option "Add Python to PATH".
  3. Verify the installation by opening the terminal and typing:python --version

Variables and Data Types

In Python, variables do not need explicit declaration. Python's dynamic typing allows you to assign a variable without specifying its type.

name = "Alice" # string age = 25 # integer is_student = True # boolean

Python supports the following basic data types:

  • int: for integers
  • float: for floating-point numbers
  • str: for strings
  • bool: for boolean values
Control Flow

Python uses indentation to define the structure of control flow statements such as if, for, and while.

If Statements

age = 18 if age >= 18: print("You are an adult.") else: print("You are a minor.")

Loops

# For loop example for i in range(5): print(i) # While loop example count = 0 while count < 5: print(count) count += 1
Functions

Functions are a way to encapsulate code that performs a specific task. Python uses the def keyword to define functions.

def greet(name): return f"Hello, {name}!"   print(greet("Alice"))
Classes and Objects

Python supports object-oriented programming (OOP), allowing you to model real-world entities using classes and objects.

class Dog:
    def __init__(self, name, breed):
        self.name = name
        self.breed = breed
        
    def bark(self):
        print(f"{self.name} says woof!")
    
# Creating an object
dog1 = Dog("Buddy", "Golden Retriever")
dog1.bark()
Inheritance

Inheritance allows you to create a new class that inherits attributes and methods from another class.

class Animal:
    def __init__(self, name):
         self.name = name
    
    def speak(self):
        print("Animal sound")
    
class Cat(Animal):
    def speak(self):
        print(f"{self.name} says meow!")
    
cat1 = Cat("Whiskers")
cat1.speak()
Importing Modules

Modules in Python are simply Python files with .py extension. You can import and reuse code from other modules using the import statement.

import math
print(math.sqrt(16))  # Output: 4.0
Creating a Package

Packages are directories that contain multiple modules and an __init__.py file. The __init__.py file ensures that the directory is treated as a package.

Popular Libraries

Some widely used Python libraries include:

  • NumPy: for numerical computations.
  • Pandas: for data manipulation and analysis.
  • Matplotlib: for data visualization.
Error Handling

Python handles errors through exceptions. Common exceptions include ValueError, TypeError, and FileNotFoundError.

Try-Except Block

The try-except block allows you to handle errors gracefully without crashing the program.

try:
    result = 10 / 0
except ZeroDivisionError:
    print("You can't divide by zero!")
Conclusion

Python is a versatile and powerful language that is easy to learn and highly adaptable. Its wide range of libraries and frameworks make it suitable for various fields such as web development, data analysis, artificial intelligence, and automation.
